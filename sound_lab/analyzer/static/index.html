sound_lab — 楽器音アナライザ & 編集スタジオ

楽器の単音を渡すと、ADSR・倍音・残差ノイズ・非調和性・ビブラート/トレモロまで解析し、 その場で鳴らしながら音色を作り込めます。金属楽器・弦楽器向けの補正、単音クオリティ向上、複数音源の音量バランス調整と ZIP / 音色パック書き出しに対応しています。

解析サーバが必要です

このページは 解析用 Python サーバ経由で開く必要があります。HTML を直接ダブルクリックしても動きません。
① かんたん: sound_lab/analyzer/start.command をダブルクリック（初回は依存インストールで数分）。自動でブラウザが開きます。
② ターミナル: cd sound_lab/analyzer && python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt && python app.py
起動できたら http://127.0.0.1:5005 を開いてください（今このページが ならそれが原因です）。
※ 解析は不要だが「スタジオで音を作ってみたい」だけなら、下の 内蔵デモ音色で試す でもサーバなしで触れます。

指定した楽器に合わせて、基音探索と初期音色補正を調整します。
🎵 音源ファイルをドロップ / クリックして選択
.wav 推奨 ・ .flac / .ogg / .mp3 / .aiff / .m4a 可 ・ 〜32 MB ・ 1音だけを渡してください
🎚 複数の音源ファイルをまとめて選択
最大12ファイル ・ 各ファイルは単音推奨 ・ 解析後に音量バランスを調整して ZIP / 音色パック / 代表音色JSONで書き出します
C4 (60) 鍵盤か PC キーを押すと音が起動します
🎹 鍵盤クリック or PC キー（z s x d c v g b h n j m ／ 上段 q 2 w 3 e r 5 t 6 y 7 u i）を押している間鳴ります。↑↓ でオクターブ移動。Space で全停止。
