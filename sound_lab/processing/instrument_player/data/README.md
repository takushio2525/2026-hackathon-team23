# data/ — インストゥルメント定義の置き場

Processing スケッチが起動時にスキャンするフォルダ。**ここに入れた `*.json` を全部読み込み**、
スケッチ画面下の一覧から（または `[` / `]` キーで）音色を切り替えられる。

- `sound_lab/analyzer` で解析してダウンロードした `<名前>.instrument.json` を、**そのまま**ここに
  コピーするだけでよい（ファイル名は何でも可。複数置ける）。
- スケッチ実行中に JSON を足したら `r` キーで再スキャン。解析し直して上書きした場合も `r` で反映される。
- `instrument.json` という名前のファイルがあれば優先して最初に開く（無ければ `example_organ.json`、
  それも無ければ名前順で先頭）。
- `example_organ.json` … 起動確認用サンプル（手書き。解析結果ではない）。消しても構わない。
- `data/` 以外の場所にある JSON を一時的に開きたいときは、スケッチ実行中に `o` キー。

> このフォルダの `*.json`（`example_organ.json` を除く）は `.gitignore` で除外しているので、
> 各自が置いた解析結果はコミットされない。
