## 一言でいうと

**指揮棒を振る速さでテンポを操作し、5台の楽器マイコンとPCで合奏するシステム**です。
音楽経験がなくても、指揮を振るという直感的な操作で演奏に参加できます。

## 現在の構成

- 指揮者 `node_01`：XIAO ESP32-S3 Sense + GY-521（MPU6050）
- 金管 `node_02〜05`：Arduino UNO R4 WiFi × 4
- ドラム `node_06`：Arduino UNO R4 WiFi × 1
- PCアプリ：Processing 4 + Minim
- 曲：かえるのうた（32拍）を4声で輪唱、ドラムは56拍の全サイクルを担当

## できること

1. IMUの向きに依存しにくい拍検出
2. 指揮の拍間隔からBPMを推定し、演奏速度へ反映
3. UDPマルチキャストとマスタ時刻指定による同期演奏
4. 自由演奏と、100 BPMを維持して得点を競うゲームモード
5. 倍音JSONを使った金管の加算合成と、専用ドラム合成
6. 指揮者の状態に追従するメニュー・演奏・結果画面

## 評価目標

中心となる評価目標は、楽器間の発音タイミング差を **20 ms以内**に収めることです。
ほかに拍検出率、通信遅延、楽譜一致、音階精度、起動時間などを検証します。
