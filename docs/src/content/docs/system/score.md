## 金管4声

`node_02〜05`は同じ32拍の「かえるのうた」楽譜を持ち、開始位置だけをずらします。

| ノード | 音色 | 頭の休符 |
|---|---|---:|
| node_02 | トランペット | 0拍 |
| node_03 | ホルン | 8拍 |
| node_04 | トロンボーン | 16拍 |
| node_05 | チューバ | 24拍 |

サイクル長は`32 + 24 = 56`拍です。最後の声部が1周を終えるまで、先頭声部は次の周回を始めません。

## `ScoreEvent`

1要素が1拍を表します。

```cpp
struct ScoreEvent {
  uint16_t beatAt;
  uint8_t noteNumber, velocity;
  uint16_t durationQ8;
  uint8_t flags;
  uint8_t subNote, subVelocity;
  uint16_t subOffsetQ8, subDurationQ8;
};
```

`durationQ8`は256で1拍です。`sub*`は8分音符など拍の途中の音を予約するために使います。

## ドラム

node_06は56拍の専用譜を持ちます。基本はキック、声部の入りと区切りでクラッシュ、
終盤にスネアフィルを入れます。GM番号36=キック、38=スネア、49=クラッシュです。

## 途中参加と欠損回復

楽譜位置はローカルな加算ではなく、受け取った`beatNo`から毎回計算します。
そのためPCの途中起動や1拍の欠損後も、次のBEATで現在位置へ戻れます。
