test_v1は、指揮者と楽器3台のUDP同期を最小構成で成立させるための検証版です。
C major圏の和音を拍ごとに鳴らし、通信・時刻・発音経路を先に確認しました。

## 残している理由

- productionよりコード量が少なく、同期の核を追いやすい
- test_v2以降で増えた楽譜・音色・ゲーム処理との差分を確認できる
- 初期の設計判断と実験結果を再現できる

## productionとの違い

- 楽譜による輪唱やゲームモードがない
- NOTE形式とPCアプリの前提が異なる
- 現在の台数、音色、状態値、UIパケットに対応しない

test_v1のファームとproductionのPCアプリを混在させないでください。
