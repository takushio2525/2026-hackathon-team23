## 金管

金管は音色JSONに定義された倍音を合成します。基本式は次の形です。

```text
y(t) = Σ amp[k] × sin(2π × f0 × ratio[k] × t + phase[k]) × envelope(t)
```

基音`f0`はMIDIノートから求め、楽器ごとのオクターブ補正と音量補正を適用します。
ADSRまたは実測包絡、非調和性、ビブラート、トレモロ、整形ノイズを組み合わせます。

## ボイス管理

- 最大同時発音数：24
- NOTEの`durationMs`で自動リリース
- 上限を超える場合は古いボイスから解放
- SpaceキーまたはUIタイムアウトで全停止

## ドラム

`instrumentId >= 4`は金管とは別の`DrumEngine`へ送ります。
GMドラムの`noteNumber`でキック、スネア、ハイハット、クラッシュを判別し、専用合成または録音サンプルを使います。

音が割れる場合は、個別音色だけでなく4声とドラムを合算したマスター音量を確認してください。
