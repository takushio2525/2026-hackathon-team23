## フレーム受信

各ポートは受信バッファからmagicと20 B長を探し、完全なフレームだけを共有キューへ渡します。
未知typeや不正versionは発音・画面更新に使いません。

## 役割の自動判定

- UIパケットを受信：メイン操作UI
- NOTEの`partId=0x02`を受信：メイン操作UI
- その他のNOTEだけを受信：アナライザ

1アプリで複数ポートを開けます。node_02を含むアプリはメインUIとして全ポートのNOTEを鳴らせます。

## 画面

| 条件 | 画面 |
|---|---|
| ポートなし | ポート選択 |
| 接続済み・状態待ち | 待機 |
| Menu | モード選択 |
| Conducting + mode 0 | 自由演奏 |
| Conducting + mode 1 | ゲーム演奏 |
| Result | 結果 |
| アナライザ役 | 波形・受信状況 |

UIが2秒途絶えるとマスター再起動と判断し、全音を止めて待機へ戻ります。
