title: Processingを動かす

## 起動

Processing 4で次を開きます。

```text
pc_app/production/orchestra_resynth/orchestra_resynth.pde
```

Minimライブラリが必要です。メインスケッチと同じフォルダにある共通タブは
`pc_app/common/`へのシンボリックリンクなので、リポジトリ構造を崩さず開いてください。

## ポート選択

- `r`：再列挙して画面をリセット
- `f`：USBポートだけ表示するフィルタを切り替え
- 一覧をクリック：ポートを開閉

複数ポートを1つのアプリで開けます。UIパケット、または`partId=0x02`を受信するとメインUI、
ほかの楽器NOTEだけを受けるとアナライザになります。

## 操作キー

| キー | 動作 |
|---|---|
| `t` | テスト和音 |
| `0`〜`3` | 対応する金管音色でC4を発音 |
| `a` | 実測包絡と単純ADSRを切り替え |
| `+` / `-` | マスター音量 |
| Space | 全音停止 |
| `i` | 音色定義を再読込 |

シリアルモニタとProcessingは同じUSBポートを同時に開けません。
