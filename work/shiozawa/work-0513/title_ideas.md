# プロジェクト題名 候補（日本語系）

コンセプト: **指揮ジェスチャーを IMU で検出し、複数の ESP32 ノードを同期させて輪唱・合奏するシステム**
（指揮者役デバイス = XIAO ESP32-S3 Sense + GY-521 が腕の振りから拍・テンポを検出 → 演奏ノードがそれに同期）

第一候補は今のところ **エアタクト（AirTact）**。以下は日本語寄りの対案リスト。

---

## A. カタカナ造語系（「エア＋音楽用語」「タクト＋同期」路線）

| 候補 | 読み / 英表記 | 由来・ニュアンス | 想定サブタイトル |
|---|---|---|---|
| **エアタクト** ◎現本命 | AirTact | 「エアで指揮棒（タクト）を振る」が直感的に伝わる。短く言いやすい | 振るだけで、みんなで奏でる ― ジェスチャー同期アンサンブル |
| エアコンダクト | AirConduct | 「エア指揮」をそのまま英語化。意味は明快だがやや説明的 | 腕の振りで合奏をコントロールする分散演奏システム |
| タクトリンク | TactLink | 指揮（タクト）でノードが繋がる（リンク）。同期・連携の感じが出る | 指揮の拍で全ノードがつながる合奏システム |
| タクトシンク / シンクタクト | TactSync / SyncTact | 「指揮 × 同期」を直球で。技術寄りの響き | IMU で拍を検出し、複数デバイスをリアルタイム同期 |
| タクトウェーブ | TactWave | 指揮 × 波（手を振る wave / 同期が広がる wave）のダブルミーニング | 一振りが波のように全ノードへ伝わる合奏システム |
| エアバトン | AirBaton | 指揮棒（バトン）を空中で。バトンには「受け渡し＝輪唱」の含意も | 指揮棒を空に振って、声部をリレーする輪唱システム |
| シンフォニア・エア | Synfonia Air | sync + symphonia。「同期する交響」 | 指揮ジェスチャーで奏でる分散シンフォニー |
| ユビフリ / ウデフリ | YubiFuri / UdeFuri | 「指振り／腕振り」をそのままカタカナに。親しみやすく和テイスト | 腕を振るだけで、みんなで合奏 |
| フリフリオーケストラ | FuriFuri Orchestra | 「振り振り」のオノマトペ。デモ受けする軽さ。発表で掴みやすい | 腕を振ると、デバイスたちがオーケストラになる |

## B. 漢字熟語・造語系（硬め・ロゴ映え重視）

| 候補 | 読み | 由来・ニュアンス | 想定サブタイトル |
|---|---|---|---|
| **振奏** | しんそう | 「振る」＋「奏でる」の造語。二字で締まる。報告書・スライド映え | ―指揮ジェスチャーで奏でる分散合奏システム― |
| 揮奏 | きそう | 「指揮」の「揮」＋「奏」。"演奏"と韻を踏む造語 | 指揮の動きが、そのまま演奏になる |
| 同振 | どうしん | 「同期」＋「振り」。"同心"ともかかる（心を合わせる） | 振りを合わせ、音を合わせる ― 分散同期アンサンブル |
| 指揮同期 | しきどうき | 説明そのもの。何のシステムか一発で伝わる（その分ひねりは無い） | IMU 指揮検出による複数デバイス合奏システム |
| 合図合奏 | あいずがっそう | 「合図（指揮）」で「合奏」。語呂が良く、意味も通る | 指揮者の合図ひとつで、みんなが鳴る |
| 一振合奏 / 一振り合奏 | ひとふりがっそう | 「一振りで合奏が始まる／揃う」。タイトルに動きがある | 腕の一振りが、全ノードの演奏になる |
| 振指（または 振り指揮） | ふりしき | 「振って指揮する」を縮めた造語 | ジェスチャー指揮で同期する分散演奏システム |

---

## C. 和語・フレーズ系（やわらかい・コピー寄り）

これらは「キャッチコピーをそのまま題名にする」路線。サブタイトル無しでも成立する。

- **ふって、そろえて、かなでる** … プロセス（振る→同期→演奏）を 3 語で。やわらかく親しみやすい
- ひと振りで、みんな鳴る … 短い。指揮の "1 アクション → 全体" を端的に
- 腕で奏でる合奏 … シンプル。何をするかが分かる
- みんなでタクト … 「指揮を全員で共有して合奏する」雰囲気。軽い
- 指揮、はじめる。… 余白のあるコピー。スライド表紙向き
- 振れば、つながる。… 同期（つながる）に寄せたコピー

---

## D. ハイブリッド表記（題名＋ふりがな／英名併記）

実際の提出時はこの形にすると一番収まりが良い：

- `エアタクト（AirTact）― 指揮ジェスチャーで複数ノードを同期させる合奏システム`
- `振奏（しんそう）/ AirTact ― ジェスチャー指揮による分散同期アンサンブル`
- `エアタクト ― ふって、そろえて、かなでる`

---

## おすすめの絞り込み

1. **エアタクト（AirTact）** … 万人に伝わる・言いやすい・デモ映え。本命のままで良い。
2. **振奏（しんそう）** … 「日本語らしさ・締まり」を取るならこれ。報告書/スライドの見た目が引き締まる。英名 `AirTact` を併記すれば両取り可能。
3. **ふって、そろえて、かなでる** … サブタイトル不要のコピー型。発表で掴みを作りたいとき。

→ 提案: **題名「エアタクト（AirTact）」＋ 漢字を当てたいなら副題やロゴに「振奏」を添える**、の二段構えが一番強い。
