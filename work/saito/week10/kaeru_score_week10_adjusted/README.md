# kaeru_score_week10_adjusted

week9 の `kaeru_score_debug` をもとにした、week10 用の修正版 Processing スケッチ。
フルートとオルガンの音色を追加し、それぞれ自然に聞こえやすい音域で演奏する。

## 変更点

- ドラムパートを 4 分の 4 拍子として整理
  - 1・3 拍目: キック
  - 2・4 拍目: スネア
  - 各声部の入り（0 / 8 / 16 / 24 拍）と最後の拍だけクラッシュ + キックで強調
- 4声の担当をフルート / トランペット / トロンボーン / オルガンに変更
  - フルート: C5〜A5。`tone_sample` を使った原音サンプル主導の再生
  - トランペット: C5〜A5
  - トロンボーン: C3〜A3
  - オルガン: C3〜A3。低めの支えとして使用
- 全パートの音量を week9 より大きめに調整
  - `MASTER_GAIN = 1.35`
  - 各声部の `amplitude` をフルート / オルガン入りの構成に合わせて調整
  - ドラムは全体とのバランスを見て `DRUM_AMPLITUDE = 0.095` に微調整
  - フルートは少し前に出るよう、`amplitude = 0.28`、`FLUTE_SAMPLE_GAIN = 1.32` に調整
- 旋律楽器とドラムの聴感上のズレ対策
  - 拍位置は変更しない
  - 倍音合成の旋律楽器は `BRASS_ATTACK_SCALE = 0.45` で立ち上がりを速くする
  - フルートは `tone_sample` のループ再生を使い、完成版の質感を保つ

## 実行方法

Processing 4 で `kaeru_score_week10_adjusted.pde` を開き、Run する。

- `P`: 全パート再生
- `1`〜`4`: 旋律各パートを単独再生
- `5`: ドラムを単独再生

音色 JSON はこのフォルダ内の `data/` に同梱しているため、外部フォルダへの参照は不要。
