# かえるのうた 4声輪唱・音域調整版

week7 の仮スケッチをもとに、金管4パートが同じ `MELODY_SCORE` を使って「かえるのうた」を輪唱する Processing 4 用の確認スケッチです。

| パート | 入り拍 | 基準譜からの移調 | 演奏音域 |
|---|---:|---:|---|
| トランペット | 0 | +1 オクターブ | C5〜A5 |
| ホルン | 8 | 変更なし | C4〜A4 |
| トロンボーン | 16 | -1 オクターブ | C3〜A3 |
| チューバ | 24 | -2 オクターブ | C2〜A2 |

チューバは低音伴奏ではなく、24拍遅れで入る4声目の主旋律です。そのため、全体は56拍（32拍の譜面 + 24拍の入り遅れ）になります。

ドラムは week7 の `0.16` から `0.075` へ音量を下げ、各拍の裏に控えめなハイハットを追加しました。再解析済みのキック、スネア、ハイハット、クラッシュは、各JSONの `drum_sample`（解析した原音1打）を Minim の `AudioSample` で再生します。サンプルのないJSONだけは、従来の倍音・ノイズ・ADSRによる合成へ自動的に戻ります。最後の4拍には短いフィルを加えています。

## 音色データ

必要な音色 JSON はすべてこのフォルダの `data/` に同梱しています。ドラムを原音再生するには、各ドラムJSONに `drum_sample` が含まれている必要があります。

## 実行方法

1. Processing 4 で `kaeru_score_debug.pde` を開く。
2. Contribution Manager から `Minim` を導入する（未導入の場合）。
3. Run 後は再生待機状態になる。`P` で全パートを再生、`1`〜`5` で各パートを単独再生できる。
