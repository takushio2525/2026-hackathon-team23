# basic_detail_design

## このフォルダの目的

齋藤翔太の担当箇所について、作成済みの `system_design_report/` をもとに基本設計・詳細設計として提出しやすい形に整理した資料を置くフォルダです。

対象は WBS 116、128、129、130 です。楽譜データ形式、4パート振り分け、音高・拍の表現、Arduino から PC へ送る NOTE データ仕様を扱います。

## ファイル

- `plan_25G1053.tex`: 基本設計・詳細設計の本文
- `plan_25G1053.pdf`: コンパイル済みPDF
- `hack-cover.sty` / `hack-fonts.sty` / `hira-stix.sty`: LaTeX 用スタイルファイル

## 不要になったら

提出版に統合済みで参照しなくなった場合は削除して構いません。
