# system_design_report

## このフォルダの目的

`work/saito/week2/report/` の内容をコピーし、レビュー結果とシステム設計書ルーブリックに合わせて作り直した、齋藤担当範囲の仕様書を置くフォルダです。

対象は WBS 116、128、129、130 です。楽譜データ形式、4パート振り分け、音高・拍の表現、Arduino から PC へ送る NOTE データ仕様を扱います。

## ファイル

- `plan_25G1053.tex`: 仕様書本文
- `plan_25G1053.pdf`: コンパイル済みPDF
- `flow1.png`: week2 レポートからコピーした図。今回版では参照していませんが、比較用に残しています。
