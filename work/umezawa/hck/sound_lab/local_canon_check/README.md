# local_canon_check

Arduino・シリアル・UDPを使わず、PC 1台の単一タイムラインで production と同じ
「かえるのうた」4声輪唱と56拍ドラムを鳴らす切り分け用スケッチです。

## 起動

Processing 4で `local_canon_check.pde` を開いて Run します。Minimライブラリが必要です。

| キー | 動作 |
|---|---|
| `P` | 金管4声＋ドラムを曲頭から再生 |
| `1`〜`4` | Trumpet / Horn / Trombone / Tubaを単独再生 |
| `5` | Drumを単独再生 |
| `Space` | 停止 |
| `+` / `-` | 音量調整 |

固定テンポは100 BPMです。金管は0 / 8 / 16 / 24拍から入り、ドラムは輪唱サイクル全体の
56拍を演奏します。再生は1サイクルで終了し、`P`を押すと最初から再生します。

## 確認方法

- このスケッチで揃う場合: 音色と楽譜は揃っているため、Arduino間の通信・クロック同期・受信処理を確認する
- このスケッチでもドラムだけズレて聞こえる場合: 金管音色のattackやドラム原音の無音区間など、聴感上の立ち上がりを確認する

音色合成タブは、Processing IDEで確実に読み込めるよう `pc_app/common/` の同等コードを
スケッチ内に配置しています。音色JSONは
`pc_app/production/orchestra_resynth/data/` へのシンボリックリンクです。
